# 奖惩信息管理功能使用指南

## 功能概述

教师管理平台的奖惩信息管理功能允许教师为所管理的学生记录、编辑、删除奖惩信息，支持奖励和惩罚两种类型的完整管理。

## 功能特性

### 1. 奖惩类型支持
- **奖励类型**：奖学金、荣誉称号、竞赛获奖、论文发表、专利申请、社会实践等
- **惩罚类型**：纪律处分、警告、严重警告、记过、留校察看等

### 2. 级别分类
- 班级级别
- 院系级别
- 校级
- 省级
- 国家级

### 3. 完整的CRUD操作
- ✅ 新增奖惩记录
- ✅ 编辑奖惩信息
- ✅ 删除奖惩记录
- ✅ 查看奖惩详情

### 4. 高级功能
- 📊 统计信息展示
- 🔍 多维度筛选
- 📅 按年份分组展示
- 📱 响应式界面设计

## 使用步骤

### 1. 查看学生奖惩信息

1. **访问学生列表**
   - 登录教师管理平台
   - 点击左侧菜单"我的学生"

2. **进入学生详情**
   - 在学生列表中点击"查看档案"按钮（眼睛图标）
   - 或直接点击"奖惩记录"按钮（奖杯图标）

3. **查看奖惩信息**
   - 在学生详情页面点击"奖惩信息"标签页
   - 系统将显示该学生的所有奖惩记录

### 2. 新增奖惩记录

1. **打开新增界面**
   - 在奖惩信息页面点击"新增奖惩"按钮

2. **填写基本信息**
   ```
   奖惩类型：选择奖励或惩罚
   奖惩名称：填写具体的奖惩名称
   级别：选择奖惩的级别（班级/院系/校级/省级/国家级）
   分类：选择具体的分类（如奖学金、纪律处分等）
   详细描述：详细描述奖惩的原因、经过等
   发生日期：选择奖惩发生的日期
   ```

3. **保存信息**
   - 填写完成后点击"保存"按钮
   - 系统将自动保存并刷新列表

### 3. 编辑奖惩记录

1. **选择要编辑的记录**
   - 在奖惩列表中找到需要编辑的记录
   - 点击右侧的"编辑"按钮（铅笔图标）

2. **修改信息**
   - 在弹出的表单中修改相应信息
   - 注意：奖惩类型不可修改，如需更改请删除后重新添加

3. **保存修改**
   - 点击"更新"按钮保存修改

### 4. 删除奖惩记录

1. **选择要删除的记录**
   - 在奖惩列表中找到需要删除的记录
   - 点击右侧的"删除"按钮（垃圾桶图标）

2. **确认删除**
   - 系统将弹出确认对话框
   - 确认无误后点击"确认删除"

3. **完成删除**
   - 系统将删除该记录并自动刷新列表

### 5. 筛选奖惩信息

1. **使用筛选功能**
   - 在奖惩信息页面上方可以看到筛选选项
   - 可按以下条件筛选：
     - 类型：奖励/惩罚
     - 级别：班级/院系/校级/省级/国家级
     - 状态：待审核/已通过/已驳回

2. **查看筛选结果**
   - 选择筛选条件后列表将自动更新
   - 可以同时使用多个筛选条件

## 界面说明

### 奖惩统计卡片
- **绿色卡片**：显示奖励记录数量
- **红色卡片**：显示惩罚记录数量  
- **蓝色卡片**：显示最高级别奖项

### 奖惩记录卡片
- **奖励记录**：绿色背景，左侧绿色边框
- **惩罚记录**：红色背景，左侧红色边框
- **状态标签**：显示审核状态
- **级别标签**：显示奖惩级别

### 按年分组展示
- 系统自动按年份对奖惩记录进行分组
- 同一年份内按时间倒序排列
- 奖励和惩罚记录分别展示

## 注意事项

### 1. 数据准确性
- 请确保填写的奖惩信息真实准确
- 奖惩记录将纳入学生综合素质评价
- 建议在添加前核实相关信息

### 2. 及时性
- 奖惩发生后应及时录入系统
- 避免信息遗漏或延迟
- 定期检查数据的完整性

### 3. 权限管理
- 只有教师可以为学生添加奖惩记录
- 学生可以查看自己的奖惩记录
- 管理员可以管理所有奖惩数据

### 4. 数据安全
- 重要奖惩记录建议做好备份
- 删除操作不可恢复，请谨慎操作
- 定期导出重要数据

## 常见问题

### Q1: 如何批量导入奖惩信息？
A: 目前系统支持单条录入，批量导入功能正在开发中。

### Q2: 奖惩记录可以修改哪些信息？
A: 除奖惩类型外，其他信息都可以修改，包括名称、级别、描述、日期等。

### Q3: 删除的奖惩记录能恢复吗？
A: 删除操作不可恢复，请在删除前仔细确认。

### Q4: 如何查看学生的奖惩趋势？
A: 在奖惩信息页面可以查看按年份分组的统计信息，了解学生的表现趋势。

```typescript
interface RewardPunishment {
  id: string;
  student_id: string;
  type: 'reward' | 'punishment';
  name: string;

  description: string;

```

### API接口
- `GET /api/reward-punishments/student/:id` - 获取学生奖惩列表
- `POST /api/reward-punishments` - 创建奖惩记录
- `PUT /api/reward-punishments/:id` - 更新奖惩记录
- `DELETE /api/reward-punishments/:id` - 删除奖惩记录
